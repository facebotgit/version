FaceBOT

Guia de resolução de problemas

Porque o FaceBOT apresenta erros?

Existem muitos motivos que causam erros no programa. Alguns deles podem ser:

1.Configurações erradas ou incorretas

2.Componentes não instalados, corrompidos, ou incompatíveis

3.Incompatibilidade com o sistema

4.Falha em algum componente

5.Manuseio do navegador enquanto anúncios são publicados

Correção de problemas

1.Configurações erradas ou incorretas

Este é um problema bem comum. É simples de resolver:

a.Na primeira página (Ad), verifique a programação de dias e horários. O programa irá publicar os anúncios nos dias semenais que estiverem marcado verde. Para marcar ou desmarcar, basta clicar no dia. A publicação será feita nos horários digitados no campo “Horário para anunciar”. Certifique-se que os horários seguem o seguinte formato: HH:MM, isto é, dois algarismos para a hora e dois algarismos para o minuto, ex.: 09:00. Para adicionar mais de um horário, digite um ponto e vírgula para separar os horários, ex.: 09:00;11:30;14:00.

b.Na segunda página, verifique se as informações de contas do Facebook estão corretas. Certifique-se que o Email e Senha da conta correspondem ao login correto do Facebook. Na parte

“Configurações”, verifique se as configurações estão corretas. No campo “Grupos por anúncio” deve haver um número entre 1 e 750. Será a quantidade de vezes que um único anúncio será publicado. Após digitar o horário tecle Enter ⏎ para garantir que o(s) horário(s) seja(m) salvo(s).

ATENÇÃO: Usar um valor muito alto, dependendo da quantidade de horários e anúncios, pode resultar no bloqueio da conta do Facebook.

Certifique-se que a pasta selecionada no campo “Pasta de anúncios” existe. É nesta pasta que o programa guarda os anúncios e suas configurações. No campo “Grupo para

começar” deve haver uma url de um grupo do Facebook em que todas as contas usadas pelo programa devem participar.

c.Clicando em “Editar/Apagar/Criar anúncios” (na página “Ad”) deve aperecer sua lista de anúncios. Certifique-se que todos eles contém estas informações: Título (primeiro campo); preço (segundo campo); localização (terceiro e maior campo). Caso alguma informação esteja faltando, clique em “Editar” e insira o que for nescessário.

2.Componentes não instalados, corrompidos, ou incompatíveis

Este problema geralmente ocorre quando o navegador Mozilla Firefox não abre ao anunciar. Para resolver:

a.Verifique se o Firefox está instalado. Para fazer isso, tecle  e procure por Firefox. (Caso não encontre o programa Firefox, quer dizer que ele não está instalado. Instale-o).

b.Instale o geckodriver correto. Este é um componente muito importante usado pelo FaceBOT ao anunciar. Verificar se ele está instalado na versão certa é um pouco complicado, mas para garantir, siga este procedimento:

1.Desinstale o Firefox

2.Instale a versão de 32-bits. Baixe-a neste link: https://drive.google.com/open?id=104UWHb9FUV47buEaR w8_v9AdSzQ6ElMU

3.Na primeira página do programa (Ad), clique em “Parar anúncios”. Após isso, tecle  + R (ou abra o Prompt de Comando) e digite: taskkill /im geckodriver.exe /f e clique em Ok.

4.Verifique a versão do FaceBOT. Para isso, vá para a primeira página (Ad) > Configurações do Software > Programa. Estará escrito o nome da versão (Geralmente algo como 2.2, 3.0,

...).

5.Instale o geckodriver. Baixe-o neste link: https://github.com/facebotgit/version/raw/master/Driver/W in32/geckodriver.exe. Após baixar, entre na pasta Documentos > FaceBOT_(nome da versão) > Release e mova o geckodriver para esta pasta. Caso nescessário, clique em

“Substituir”.

6.Reinicie o computador.

c.Verifique se o python está instalado. Para fazer isso, tecle  + R e digite: cmd.exe (Clique em Ok). Abrirá um programa chamado

“Prompt de comando” ou “cmd”. Nele, digite python -V e tecle Enter. Deverá aperecer algo como: Python 3.7.6. Caso não

apareça, baixe e instale o Python neste link: https://drive.google.com/open?id=1FI5cwOAV7g8jw0DylsX5hpL FIDq2NHd_.

Certifique-se de marcar esta opção ao instalar o Python:

Após instalar o python, baixe e execute (basta clicar no arquivo) este arquivo: https://drive.google.com/open?id=1NkxPdIWD2f2GGNhplJAyMNCp Wg8LAmNh.

3.Incompatibilidade com o sistema

Assim como todo programa, o FaceBOT precisa de recursos, tanto de hardware quanto de software para ser executado sem problemas. Confira os rqeuisitos mínimos aqui: https://facebotgit.github.io/site/index.html#requisitos. O FaceBOT pode até funcionar em computadiores abaixo dos requisitos, mas podem ocorrer vários problemas. Não é recomnedável instalar o FaceBOT em uma máquina que não atende os requisitos mínimos. Para verificar se seu computador é compatível com os requisitos de hardware, siga este procedimento (Apenas para Windows 8 ou superior): Tecle  + R (ou abra o Prompt de Comando) e digite: taskmgr.exe (Clique em Ok). Vá para a aba Desempenho > CPU.

Verifique os campos “Velocidade base” e “Núcleos”. Velocidade base deve ser maior ou igual a 1.6 GHz, enquanto núcleos deve ser maior ou igual a 2. Após isso clique em Memória. No canto superior direito irá aperecer a quantidade de memória do seu computador. Deve ser maior ou igual a 2.0 GB. O resto dos requisitos podem ser instalados.

4.Falha em algum componente

Neste caso, o problema não é no FaceBOT, mas em algum componente ou serviço de terceiros, como o Facebook. Por exemplo, se o Facebook não estiver carregando como deve ou estiver com alguma falha, o FaceBOT não consegue publicar anúncios. Felizmente, falhas em produtos de grandes empresas são corrigidas em pouco tempo.

5.Manuseio do navegador enquanto anúncios são publicados

Durante a publicação dos anúncios, o Firefox deve permanecer aberto. Não use ele enquanto o FaceBOT publica os anúncios, deixe ele “trabalhar” sozinho.

6.Outros problemas

a.Caso tenha seguido todos os procedimentos e o FaceBOT ainda apresenta problemas, clique em “Estou com problemas” na primeira página do programa. (Caso apareça uma janela perguntando se deseja enviar os arquivos de relatório, clique

“Sim”, pois assim o desevolvedor pode analisar os erros e resolve-los, desde que seja um problema relacionado ao programa). Isso pode resolver alguns problemas.

b.Sistema lento. Ainda que seu sistema atenda aos requisitos mínimos, pode ficar lento se muitos programas estiverem abertos. Antes de anunciar, feche todos os programas que não estão sendo usados. Também é recomendável desfragmentar o disco rígido caso note que seu sistema está muito lento. Um sistema lento pode impedir o carregamento do Facebook, o que impossibilita a publicação de anúncios.

c.Caso tenha feitos todos os procedimentos e não obteve sucesso, entre em contato conosco mandando um e-mail para: facebot.email@gmail.com.
